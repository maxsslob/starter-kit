What's in exactly?

  • Gulp- Automatic scss/sass/js/babel compiler/minifier & file watcher
  • Vue.js- JS Framework that combine everything good from others.
  • bulma.io- Modern CSS framework based on flexbox.
  • Font Awesome- Most popular icon font.
  • Pace- Cool, automatic page loading indicator.
  • jQuery- To save time interacting with the DOM
  • Easing- Script that adds cool animations available in some jQuery functions.

What's your first steps?

  1. Fill the folders - add images, to dist/assets/images, fonts to dist/assets/fonts or delete the directory if you don't need to store them on your server.
  2. Deside which plugins/frameworks to use and add or remove them to/from the project.
  3. Change/Write base code/test of plugins/scss variables/fonts
  4. HTML
  5. CSS
  6. JS/jQuery/Vue
