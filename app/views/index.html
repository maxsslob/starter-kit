Features

  • Gulp- Automatic scss/sass/js/babel compiler/minifier & file watcher
  • Vue.js- JS Framework installed
  • bulma.io- Modern CSS framework based on flexbox.

Starting a project

  1. Fill the folders - add images, to dist/assets/images, fonts to dist/assets/fonts or delete the directory if you don't need to store them on the server.
  2. Deside which plugins/frameworks to use and add or remove them to/from the project.
